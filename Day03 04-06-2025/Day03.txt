before performing any action (click type select ) we should find(search) the element 
to find the element --> locators--> 8 locators
tagname--> By.TAG_NAME
id-->By.ID
name-->By.NAME
class name--> By.CLASS_NAME
link  text--> By.LINK_TEXT
partial link text-->By.PARTIAL_LINK_TEXT
cssSelector
xpath

find element takes 2 arg--> locator type and locator value
if it is not able to locate the element , it will throw NoSuchElementException
